СИСТЕМНЫЕ ТРЕБОВАНИЯ
======================

ОПЕРАЦИОННАЯ СИСТЕМА:
----------------------
✅ Windows 10 (64-bit) - Версия 1903 и выше
✅ Windows 11 (64-bit) - Все версии
❌ Windows 8/8.1 - Не поддерживается
❌ Windows 7 - Не поддерживается
❌ macOS - Не поддерживается
❌ Linux - Не поддерживается

ПРОЦЕССОР:
----------
- Минимальный: Intel Core i3 или AMD Ryzen 3
- Рекомендуемый: Intel Core i5 или AMD Ryzen 5
- Частота: 2.0 ГГц и выше

ОПЕРАТИВНАЯ ПАМЯТЬ:
-------------------
- Минимальная: 4 ГБ
- Рекомендуемая: 8 ГБ
- Оптимальная: 16 ГБ

СВОБОДНОЕ МЕСТО НА ДИСКЕ:
-------------------------
- Для установки: 500 МБ
- Для работы: 1 ГБ
- Для резервных копий: 2-5 ГБ

ДОПОЛНИТЕЛЬНЫЕ ТРЕБОВАНИЯ:
----------------------------
✅ .NET 8.0 Runtime (устанавливается автоматически)
✅ DirectX 11 (входит в состав Windows)
✅ Интернет-соединение (для активации и обновлений)

ПРОВЕРКА СОВМЕСТИМОСТИ
======================

Чтобы проверить совместимость вашей системы:

1. Нажмите Win + R
2. Введите "dxdiag" и нажмите Enter
3. Проверьте версию Windows в разделе "Система"
4. Убедитесь, что у вас достаточно ОЗУ и места на диске

РЕКОМЕНДАЦИИ ПО ПРОИЗВОДИТЕЛЬНОСТИ
==================================

Для лучшей производительности:
• Закройте неиспользуемые программы перед запуском
• Регулярно очищайте диск от временных файлов
• Обновляйте Windows до последней версии
• Используйте SSD для хранения программы и базы данных

ИЗВЕСТНЫЕ ОГРАНИЧЕНИЯ
=====================
