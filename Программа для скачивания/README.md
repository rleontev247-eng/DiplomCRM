# DiplomCRM - Система управления клиентами

## 📋 Обзор программы

DiplomCRM - это система управления взаимоотношениями с клиентами (CRM) для малого и среднего бизнеса.

### 🎯 Основные функции:

**👥 Управление клиентами:**
- Добавление и редактирование клиентов
- Поиск и фильтрация
- История взаимодействий

**💼 Управление сделками:**
- Создание и отслеживание сделок
- Воронка продаж
- Управление приоритетами

**📊 Аналитика и отчеты:**
- RFM анализ клиентов
- Анализ эффективности сделок
- Индекс удовлетворенности (CSAT)
- Экспорт в Excel

**⚙️ Настройки:**
- Резервное копирование данных
- Визуальные уведомления
- Создание тестовых данных

---

## 🚀 Системные требования

**Минимальные:**
- Windows 10/11 (x64)
- 4 ГБ ОЗУ
- 500 МБ свободного места на диске

**Рекомендуемые:**
- Windows 10/11 (x64)
- 8 ГБ ОЗУ
- 1 ГБ свободного места на диске

---

## 📦 Установка программы

### Шаг 1: Распакуйте архив
Распакуйте содержимое архива в любую папку на вашем компьютере.

### Шаг 2: Установите программу
1. Перейдите в папку `MyFirstCRM_Distribution`
2. Запустите файл `MyFirstCRM_Setup.exe`
3. Следуйте инструкциям установщика

### Шаг 3: Запустите программу
После установки найдите программу в меню "Пуск" или на рабочем столе.

---

## 🔑 Первичный запуск

При первом запуске программа автоматически создаст базу данных и предложит:
- Создать тестовые данные для ознакомления
- Настроить резервное копирование
- Включить уведомления

---

## 💾 Резервное копирование

**Важно:** Программа автоматически хранит данные в:
`%LocalAppData%\MyFirstCRM\database.db`

**Рекомендации:**
- Включите автоматическое резервное копирование в настройках
- Периодически создавайте резервные копии вручную
- Храните копии на внешнем носителе
